# Utils

Um compilado de alguns scripts genéricos que podem ser úteis para diversos momentos.

## [arrayUtils](./arrayUtils.js)

Funções complementares para trabalho com arrays.

- shuffleArray (array)

Retorna o array embaralhado.

## [operators](./operators.js)

Funções que realizam operações não nativas do Javascript.

- xOr (boolean, boolean)

Retorna o `ou exclusivo` de dois booleans.
